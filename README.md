## 使用方法

请优先使用 不学习何以强国 ，它可以全自动，以后维护工作以它为主， 强国学习 需要手动进入答题。

装个浏览器插件 tampermonkey （网上很多教程），点击插件里添加按钮，去掉编辑框里原来的代码，把 不学习何以强国 js 脚本复制粘贴进编辑框保存。开启这个脚本，然后进入网页强国 www.xuexi.cn 登录，刷新登录网页，左上角有启动按钮。
